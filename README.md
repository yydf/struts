特性
-------------------------
* 轻量
* 零配置
* 自动扫描拦截器Interceptor
* 自动注入@Resource对象

环境
-------------
- JDK 7
- Servlet 3
- slf4j

如何使用
-----------------------
* 添加dependency到POM文件:

```
<dependency>
    <groupId>cn.4coder</groupId>
    <artifactId>struts</artifactId>
    <version>0.0.7</version>
</dependency>
```
